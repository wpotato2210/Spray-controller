# USAGE

Spray Controller Usage Instructions

## Operation Steps

1. Power on the system.
2. Set the run/hold switch:
   - Run: normal operation
   - Hold: stops pump; optional bypass solenoid can activate
3. Enable or disable boom sections as needed.
4. Observe LED indicators for active boom sections.

## How Rate Control Works

- Controller maintains target application rate using speed and active width
- Active spray width equals sum of ON boom sections
- Section changes alter effective width only; flow remains globally controlled

## Notes

- Run flow calibration before first operation.
- Verify safety interlocks before field use.
